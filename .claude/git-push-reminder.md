# 🚨 GIT PUSH 絶対実行ルール

## 重要事項
**コミット作成後は必ずgit pushを実行する**

## 手順
1. `git add .`
2. `git commit -m "メッセージ"`
3. **必須**: `git push origin master`

## チェックリスト
- [ ] コミット作成
- [ ] git push実行 ← **これを忘れるな！**
- [ ] GitHub反映確認

## 例外
- なし
- 絶対に忘れてはならない
- このファイルを見たら必ずgit pushを確認せよ

## 自動実行フロー
1. 変更 → add → commit → **push** → 完了
2. push無しのコミットは不完全

---
**このルールに例外はない**
